数字运算
========

## 知识点

* 学习一下Python的数字运算
* numpy

## 实战演习

~~~python
# 基础运算
a = 5
b = 3
print(a+b)
print(a-b)
print(a*b)
print(a**b)
print(a/b)
print(a//b)

# 先乘除后加减，括号优先
result = 2+3*5
print(result)

result = (2+3)*5
print(result)

# 浮点运算
# 结果的小数点位数可能是不确定的,不过请不用担心,所有语言都有类似的问题,今后会讲到小数位的处理方法
x = 0.1
y = 0.2
print(x+y)
print(x-y)
print(x*y)
print(x/y)

# 数值字符串转换
age = 25
msg = "我今年" + str(age) + "岁"
print(msg)
~~~
