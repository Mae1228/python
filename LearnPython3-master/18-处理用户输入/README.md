## 知识点

## 实战演习

~~~python
myage = input("请输入你的年龄：")
print("你的年龄是：", myage)
print("-------------------")

# 年龄判断
if int(myage) > 30:
    print("现代老年人。")
else:
    print("加油啊，年轻人！")
~~~
