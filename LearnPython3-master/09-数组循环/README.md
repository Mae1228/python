数组循环
========

## 知识点

* for循环数组
* Python特有的缩进代码

## 实战演习

~~~python
mylist = ["curry", "harden", "lebron", "durant", "kobe"]

# 数组循环
print("Start.")
for name in mylist:
    print(name) # 缩进代码
print("End.")

# 循环索引
for i, item in enumerate(mylist):
    print(i, item)
~~~
