排序算法
========

## 知识点

* 数组的排序

## 实战演习

~~~python
# 数组排序
mylist = ["curry", "harden", "lebron", "durant", "kobe"]
print(mylist)
print(len(mylist))

mylist.sort()
print(mylist)

mylist.reverse()
print(mylist)

mylist.sort(reverse=False)
print(mylist)
mylist.sort(reverse=True)
print(mylist)

# 排序函数
lang_list = ["Python", "Javascript", "RoR", "Kotlin", "Java"]
print(lang_list)
print(sorted(lang_list))
print(sorted(lang_list, reverse=True))
~~~
